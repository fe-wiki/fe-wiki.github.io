Introduction To Editing

Github doesn't allow all users to edit repositories directly, so you must fork it, make your changes, then send a pull request.

Forking

By clicking on edit page on any page, you will be sent to a page that allows you to fork the repository (or allows you to edit in your forked repository if already created).

Click on Fork this Repository to fork it.

Editing

The pages are made using HTML. Do not edit the head of the page. Once you are complete, click on Commit changes

Submitting

You will be sent to a page where you can open a pull request to the main repository, or if you don't get that, go to the main page of the repository and click on Contribute then Open pull request

Create pages

Click on 'Add file' on your forked repository or the main repository and follow the steps above. Make sure to type in nameOfPage/index.html for the file name to make it work.
