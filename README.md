# FE-WIKI
### Before you do anything, read this

To add a page, create a folder with the wanted name for the page inside the root or inside another folder for a subpage. Create a file named index.html for your page and copy this as a template:

```<head>
  <title>Unoffical FE Wiki</title>
  <script src='/topbar.js'></script>
  <link rel="stylesheet" href="/style.css">
</head>
<body>
  <h1>ExamplePage</h1>
  <p>ExampleText</p>
</body>
```

### Making a good page

- Do not spread hate.
- State the facts.
- Link to other pages where possible.
- Do not ruin the work of others.
- Do not advertise.
- Do not use offensive words or language.
- Do not grief pages.
- Do not share personal information.
- Follow Github's Terms Of Service.
- Do not share explicit content.
- Do not link to external websites.
- Stay on topic.
